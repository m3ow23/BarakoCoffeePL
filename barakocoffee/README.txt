BarakoCoffee Language usage:

1. Direct Command Line Compilation
1.1 open cmd in barakocoffee.jar directory
1.2 type "java -jar barakocoffee.jar sample_input.bc sample_output.txt"

2. Command Line UI Compilation
2.1 open cmd in barakocoffee.jar directory
2.2 type "java -jar barakocoffee.jar" to open BarakoCoffee UI
